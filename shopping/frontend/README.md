# 前端页面使用说明

## 文件结构

```
frontend/
├── index.html          # 用户端主页面（单页应用）
├── admin.html          # 后台管理页面
├── js/
│   └── app.js          # 用户端JavaScript代码
└── README.md           # 本文件
```

## 功能说明

### 用户端（index.html）

1. **首页**
   - 商品列表展示
   - 价格筛选
   - 排序功能
   - 分页显示

2. **商品详情页**
   - 商品信息展示
   - 加入购物车
   - 立即购买

3. **购物车**
   - 查看购物车商品
   - 修改商品数量
   - 删除商品
   - 结算功能

4. **订单确认**
   - 选择收货地址
   - 确认订单信息
   - 支付订单

5. **订单列表**
   - 查看所有订单
   - 订单状态筛选
   - 支付/取消订单
   - 确认收货

### 后台管理（admin.html）

1. **商品管理**
   - 商品列表
   - 添加商品
   - 编辑商品
   - 删除商品
   - 上下架管理
   - 库存管理

2. **订单管理**
   - 订单列表
   - 订单详情查看
   - 订单发货

## 技术栈

- Vue 3（CDN方式引入）
- Element Plus（UI组件库）
- Axios（HTTP请求库）
- Vue Router（路由管理）

## 注意事项

1. **跨域问题**
   - 如果遇到跨域问题，需要在后端配置CORS
   - 或者在浏览器中禁用CORS检查（仅用于开发）

2. **Token认证**
   - 前端会自动在请求头中添加Token
   - Token存储在localStorage中
   - 如果Token过期，会自动跳转到登录页

3. **图片显示**
   - 商品图片需要完整的URL路径
   - 如果图片加载失败，会显示默认占位图

## 浏览器兼容性

- Chrome ≥90
- Firefox ≥88
- Safari ≥14
- Edge ≥90

## 开发建议

1. 建议使用本地服务器运行（如使用VS Code的Live Server插件）
2. 生产环境建议使用构建工具（如Vite、Webpack）打包
3. 建议将API地址配置为环境变量
